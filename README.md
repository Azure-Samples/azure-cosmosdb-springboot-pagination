## Setup

```bash
git clone https://github.com/Azure-Samples/azure-cosmosdb-springboot-pagination.git</br>  
cd azure-cosmosdb-springboot-pagination </br>  
mvn clean install </br>  
mvn spring-boot:run </br>  
```

## Running the sample

curl -v http://localhost:8080/api/volcanoesbypage?pageNo=0&pageSize=5 </br>
curl -v http://localhost:8080/api/volcanoesbypage?pageNo=1&pageSize=5 </br>
